
### AutoCompiler for Codeforces

`pip install html2text`

**Use:**

`./../cr.sh X.cpp` to run a problem

`./new.sh R` to create a new round

**E.g.:**

`cd ~/foo/AutoCompiler-Codeforces/500/ ; ./..cr.sh A.cpp`

`./new.sh 500`
